# 🚀 v2.8.0 — The "Mobile-First" Design Evolution

**Release Date:** May 9, 2026

---

### 📦 Release Overview

OpenShelf v2.8.0 is a visual and functional transformation focused on delivering a high-fidelity, "native app" experience for web users. Drawing inspiration from modern mobile interfaces, this update introduces fluid navigation patterns, elegant glassmorphism, and a completely reimagined discovery layout.

### ✨ Key Features & Improvements

#### 📱Smart Header

- **Scroll-to-Reveal Logic** — The header now automatically hides when scrolling down to maximize reading space and instantly reappears when scrolling up even slightly.
- **Glassmorphism & Solid States** — Implemented a dual-state header with high-end backdrop-filter blur and solid background transitions to prevent content overlap during scroll.
- **Persistent Hamburger** — Even when the main header is hidden, a minimalist menu button remains accessible in the top-right corner.

#### 🏷️ Advanced Category Navigation

- **Sticky "Glide" Bar** — The horizontal category list now smoothly glides to the very top of the screen when the header is hidden, providing one-tap filtering at all times.
- **Chips** — Clean, pill-shaped category chips with a bold "Active" state and hidden scrollbars for a clutter-free swipe experience.

#### 📚 Reimagined Book Card Component

- **Modern Geometry** — Updated cards with a soft 20px border-radius and elegant, low-opacity shadows (`rgba(0,0,0,0.04)`).
- **Floating Pill Badges** — Availability status (Available, Borrowed, Reserved) is now displayed in floating pill badges with `backdrop-filter: blur(4px)` in the top-right corner of the cover.
- **Refined Typography** — Optimized font weights and sizes for titles and authors, adding a clear "Category Tag" above each book title.

#### 🔍 Slim Integrated Search

- **Streamlined Design** — Redesigned the search bar to be slim and professional, integrating the search icon inside the input field for a more cohesive look.

#### 🎨 Premium Dark Footer

- **Vibrant Presence** — A total footer redesign featuring bold headings with highlight underlines and social icons with vibrant purple gradient backgrounds.
- **Improved Scannability** — Refined typography for links and contact info, ensuring a clean, easy-to-read finish for the site's footer.

### 🔧 Fixes & Optimizations

- **Sidebar Layering** — Fixed z-index issues where the mobile sidebar would appear underneath the fixed header.
- **Mobile Sidebar Scrolling** — Enabled native-feel vertical scrolling for the mobile navigation menu.
- **Status Bar Sync** — Added a dynamic `theme-color` meta tag to match the mobile browser's status bar with the OpenShelf header.
- **Solid UI Stability** — Transitioned to solid background headers to eliminate visual overlap of background content during scroll.
