# 🚀 OpenShelf v3.1.0 — Polish, Performance, and Stability

**Release Date:** June 30, 2026

A polished release focused on faster discovery, improved responsiveness, and stability across both public and admin experiences.

## 🌟 Highlights

### ✨ UI Refinements
- Polished book discovery, filter controls, and navigation for a smoother, more cohesive experience.
- Improved visual consistency across the public pages and mobile panels.

### ⚡ Performance Improvements
- Faster page loads and reduced friction in the infinite scroll experience.
- Optimized asset delivery across book cards and admin views.

### 🛠️ Stability Fixes
- Resolved issues in borrow/request workflows, admin reports, and mobile layout handling.
- Improved reliability of request state updates and page transitions.

### 🔒 PWA & Offline Strengthening
- Stronger offline fallback messaging and improved installable behavior.
- Better error handling for intermittent connectivity.

### 📈 Quality Enhancements
- Updated visual polish across core user-facing flows.
- Minor bug fixes and usability improvements for a more stable experience.
