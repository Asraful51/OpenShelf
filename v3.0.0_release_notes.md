# 🚀 OpenShelf v3.0.0 — Design Revamp & Enhanced User Experience

**Release Date:** May 21, 2026

A major milestone release introducing completely redesigned core layouts, a unified global color scheme powered by CSS variables, and significant new features including Book Filtering and a dedicated Settings page.

## 🌟 Highlights

### 🎨 Complete Design Revamp (V2 Layouts)
- **Profile Layout:** Reimagined the Profile page for a cleaner, modern look.
- **Footer Layout:** Modernized the website footer interface with improved spacing and typography.
- **Book Layout:** Redesigned the Book Details page and Book Cards for better interaction and visual appeal.
- **Header Layout:** Updated the navigation header for a more intuitive and aesthetic experience.
- **Modern Floating Bottom Navigation Bar:** Implemented a new floating bottom navigation bar for mobile and compact views, significantly enhancing accessibility and aesthetics.

### ⚙️ New Features
- **Book Filtering:** Added functional filtering capabilities (`Feature/filter by`) to the book discovery page to easily find specific books.
- **Settings Page:** Introduced a new dedicated settings page (`Feature/setting`) for personalized user configuration and preferences.

### 🛠️ UI Refinements & Bug Fixes
- **Unified Color Scheme:** Refactored the UI color scheme to use centralized CSS variables and replace inline/hardcoded styles for consistent theming and easier maintenance.
- **Global Color Changes:** Updated overall brand colors for a more premium feel.
- **Plus Button Fix:** Resolved the visual "jumping" problem associated with the plus/add button interactions.
